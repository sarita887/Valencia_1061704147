Bienvenidos a Nuestra Página Web

Una computadora es una máquina que puede programarse para realizar automáticamente secuencias de operaciones aritméticas o lógicas ( cálculo ).

Una computadora, también llamada ordenador o computador, es una máquina electrónica capaz de procesar datos de manera automática. Se compone de hardware (componentes físicos) y software (programas y datos). Las computadoras son herramientas versátiles que nos permiten realizar una gran variedad de tareas, desde el trabajo y el estudio hasta el entretenimiento y la comunicación.
